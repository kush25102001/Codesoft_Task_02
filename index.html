Recipe Share: A Spring Boot-Based Culinary Platform

CookBook Connect: A Recipe Sharing Platform

Description: CookBook Connect is a dynamic web application designed for food enthusiasts to share and explore a wide variety of recipes. Users can create, upload, and browse recipes, contributing to a growing community of culinary ideas. The platform supports user authentication, allowing users to manage their personal recipe collections and interact with others' submissions.

Technologies Used:

  • Spring Boot: Provides the backend services, including REST APIs, user authentication, and recipe management.
  • Thymeleaf: Facilitates dynamic rendering of HTML pages, allowing for an intuitive and responsive user interface.
  • PostgreSQL: Acts as the relational database to store user data, recipes, and other essential information securely.

CookBook Connect merges the power of Spring Boot, Thymeleaf, and PostgreSQL to deliver a seamless and robust experience for users passionate about cooking and sharing their favorite recipes.

Controlling Media Player

Controlling Media Player Using Hand Gesture

Description: This project is an innovative application that allows users to control a media player using hand gestures, eliminating the need for traditional input devices like a mouse or keyboard. The application leverages computer vision and machine learning techniques to detect and interpret hand gestures in real-time, providing a hands-free way to interact with media playback.

Technologies Used:

  • Python: Serves as the primary programming language for developing the application.
  • OpenCV: Handles real-time video capture and processing to detect hand gestures.
  • MediaPipe: A framework by Google used for hand tracking and gesture recognition, allowing the application to understand user inputs.
  • PyAutoGUI: Automates the media player controls by simulating keyboard and mouse inputs based on the detected gestures.

This project showcases the integration of computer vision and automation, offering a glimpse into the future of human-computer interaction through gesture-based controls.
